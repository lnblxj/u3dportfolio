# Unity3D Portfolio

This is a Unity3D personal portfolio project built with [Next.js](https://nextjs.org).

Quick demo: https://u3d.top

## Quick Start

This project uses [pnpm](https://pnpm.io/) as the package manager. Please ensure that Node.js and pnpm are installed locally.

1. **Install dependencies**
```bash
pnpm install
```

2. **Start the development server**
```bash
pnpm dev
```

Once started, open [http://localhost:3000](http://localhost:3000) in your browser to preview the project.
